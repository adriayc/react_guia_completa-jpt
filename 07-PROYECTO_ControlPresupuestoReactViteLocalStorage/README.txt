* Creando un proyecto React con Vite
  # npm create vite@latest
  Ingresamos o seleccionamos las siguientes opciones:
    - Proyect name
    - Select a framework
    - Select a variant
  Ahora ejecutamos los siguientes comando:
    # cd <nombre_proyecto>
    # npm install | npm i
    # npm run dev             // Levanta el proyecto
