* Creando el proyecto REDUX
  -  Crear la app
    # npx create-react-app crudredux
    # cd crudredux
    # npm start

* Instalar componentes
  - React Router Dom
    # npm i react-router-dom

  - JSON Server
    # npm install -g json-server
    + Ejecutar la db.json
      # json-server db.json --port 4000
